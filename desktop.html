Help & Features

Features

  • Schedule entries with start/end times and descriptions
  • Auto-calculate end times from next row's start time
  • Add location, notes, and URLs to events
  • Cascade AM/PM changes to all rows below
  • Preview your ICS file before downloading
  • Works with Google Calendar, Apple Calendar, Outlook, etc.

How to Use

  • Select a date using Today/Tomorrow/Choose buttons
  • Enter start time (e.g., 9:30, 1245, 8:00)
  • Add a description for each block
  • Optional: Add end time and AM/PM period
